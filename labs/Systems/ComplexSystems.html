Boids ¬∑ Flocking

local rules ‚Üí global swarm

Idea: Each ‚Äúbird‚Äù only reacts to nearby neighbors, yet the whole group flies as a coordinated flock.

For each boid we compute three steering forces:

  • Alignment: match the average direction of neighbors.
  • Cohesion: move toward the average position of neighbors.
  • Separation: steer away if neighbors are too close.

In vector form (simplified):

vnew = v + wa¬∑Falign + wc¬∑Fcohere + ws¬∑Fseparate

Where each F is a vector built from neighbors within a perception radius.

Real-world applications:

  • Designing swarm robots or drones that coordinate without a central controller.
  • Traffic and crowd simulations (how groups self-organize and jam).
  • Graphics / games: believable movement of fish schools, bird flocks, crowds.

Cellular Automaton ¬∑ Game of Life

binary grid ¬∑ simple update rule

Idea: Each cell is either alive (1) or dead (0) on a grid. The next state depends only on the 8 neighbors.

Conway‚Äôs rules (per step):

  • If a cell is alive and has < 2 neighbors ‚Üí it dies (underpopulation).
  • If alive and has 2 or 3 neighbors ‚Üí it stays alive.
  • If alive and has > 3 neighbors ‚Üí it dies (overcrowding).
  • If a cell is dead and has exactly 3 neighbors ‚Üí it becomes alive (birth).

In compact form, for cell state s ‚àà {0,1} with neighbor count n:

s' = 1 if (s = 1 and n ‚àà {2,3}) or (s = 0 and n = 3), else 0.

Real-world applications / intuition:

  • Studying how simple local rules can create persistent ‚Äúorganisms‚Äù and moving patterns.
  • Conceptual model for growth, self-repair, and self-replication.
  • Inspiration for distributed algorithms and pattern formation in materials and biology.

Predator‚ÄìPrey ¬∑ Lotka‚ÄìVolterra

population cycles ¬∑ ecosystems

Idea: Two interacting populations: prey (e.g. rabbits) and predators (e.g. foxes). Predators eat prey, and both populations change over time.

dR/dt = Œ±R ‚àí Œ≤RF
dF/dt = Œ¥RF ‚àí Œ≥F

  • Œ± ‚Äì natural prey growth rate.
  • Œ≤ ‚Äì how strongly predators remove prey.
  • Œ≥ ‚Äì predator death rate without food.
  • Œ¥ ‚Äì how efficiently eaten prey turns into new predators.

Our canvas uses agents in space with an energy model instead of continuous equations, but the qualitative behavior (cycles, booms & crashes) is similar.

  • Ecology: modeling predator‚Äìprey cycles, invasive species, or pest control.
  • Cyber / networks: attackers vs defenders, malware vs patches as predator‚Äìprey dynamics.
  • Industrial systems: load vs capacity (e.g. tasks vs worker threads / resources).
